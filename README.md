中医学以阴阳五行作为理论基础。

不过，随着社会发展。
西医崛起。
中医倒是慢慢没落了。
这种现象，倒不是说，中医不如西医。
只是说，西医更能满足现在人的要求罢了。
换句话说就是效率。
西医现在之所以占据主流市场。
就是因为西医的治疗效率要高很多。
同样的病，西医可能一天就能治好，中医却需要更多的时间。
但任何事情都具有两面性。
西医治疗效率高的同时。
自然会有一定的副作用。
比如……
隔段时间。
同样的病症会再犯。
这就是治标不治本。
但中医不同。
中医虽然治疗的时间长。
却是从切除病源的根源为出发点。
总结来说就是。
西医治病，但治的是人得的那种病。
中医治病，治疗的却是得病的那个人。
前者是治病，后者是救人！
这就是中医跟西医的区别了。
但就目前的社会发展而言。
注重根本的人，永远是少数人。
大多数人更是注重效率。
头痛医头，脚痛医脚。
根本不会考虑造成疼痛的根源到底是什么。
这也是西医慢慢占据主流市场的原因。
久而久之，中医自然是慢慢断了传承。
不过，话又说回来。
不管是中医还是西医。
其实最终目的都是为了对抗疾病。
这点倒是毋庸置疑。
